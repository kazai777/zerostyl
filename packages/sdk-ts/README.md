# @zerostyl/sdk-ts

TypeScript SDK for the [ZeroStyl](https://github.com/kazai777/zerostyl) zk toolkit on Arbitrum Stylus.

> **Status**: alpha scaffold. Type generator from `AbiSchema` is the next iteration; in-browser proving (WASM bundle) and on-chain submission helpers land after the universal verifier ships.

## Install

```bash
pnpm add @zerostyl/sdk-ts
```

## Develop

```bash
pnpm install
pnpm -F @zerostyl/sdk-ts build
pnpm -F @zerostyl/sdk-ts test
pnpm -F @zerostyl/sdk-ts lint
```

Requires Node >= 20.
